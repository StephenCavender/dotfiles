# Developer Experience Agents

Specialized AI agents focused on improving developer workflows, tooling, and productivity.

## Agents

### [DX Optimizer](dx-optimizer.md)
Developer Experience specialist improving tooling, setup, and workflows. Masters onboarding optimization, automation, and development friction reduction.

## Expertise Areas

- **Environment Setup**: Onboarding automation, dependency management, configuration
- **Development Workflows**: Task automation, build optimization, feedback loops
- **Tooling Enhancement**: IDE configuration, git hooks, CLI commands
- **Documentation**: Setup guides, troubleshooting, inline help
- **Performance**: Build/test optimization, hot reload enhancement
- **Developer Satisfaction**: Friction reduction, productivity metrics

## Common Tasks

- Streamline onboarding process
- Create development automation
- Optimize build and test times
- Configure IDE and tooling
- Create helpful documentation
- Identify and remove friction points
- Measure developer productivity
- Implement best practices

## Collaboration

DX agents often work with:
- **Engineering agents** for workflow integration
- **Ops agents** for deployment automation
- **Quality agents** for testing optimization
- **Content agents** for documentation

## Success Metrics

- Time from clone to running app
- Number of manual steps eliminated
- Build/test execution time
- Developer satisfaction scores
- Onboarding time reduction
- Tool adoption rates
